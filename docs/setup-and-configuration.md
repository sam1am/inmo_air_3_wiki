## Initial Setup

1. Power on and complete Android setup
2. Connect to WiFi
3. Sign in with Google account (international version)
4. Download recommended apps from Google Play or Aptoide
5. Update ring and touchpad firmware
6. Adjust display position and brightness
7. Install prescription lenses if ordered

---

## Developer Options

### Enabling Developer Options

1. Go to **Settings > About Device**
2. Tap **Build Number** 7 times
3. Developer Options will appear in Settings

### Key Settings

| Setting | Purpose |
|---------|---------|
| **USB Debugging** | Enable for sideloading apps |
| **ADB Commands** | Direct APK installation, advanced configs |

---

## Sideloading Apps

### Steps

1. Enable USB Debugging in Developer Options
2. Connect to PC via USB-C
3. Use ADB to install APK files:
   ```bash
   adb install path/to/app.apk
   ```
4. Popular apps available on APK sites

### Recommended Sources

- APKMirror
- APKPure
- Aptoide (built-in alternative store)

## Essential First Steps

1. **Disable auto-wake** to prevent unwanted power-on during charging
2. **Install a custom launcher** to prevent burn-in from static UI
3. **Update all firmware** for best performance
4. **Install essential apps** (see [Apps & Software](apps-and-software.md))
5. **Adjust comfort settings** (see [Comfort & Fit](comfort-and-fit.md))
