## Screen Burn-In

**Problem**: Temporary or permanent image retention on OLED display

### Prevention

- Power OFF (not standby) when charging overnight
- Use dark mode or third-party launchers
- Lower brightness below 50% for idle periods
- Minimize static UI elements

### Recovery

- Temporary burn-in usually resolves in 24 hours
- Play full-screen video to "exercise" pixels
- Lower brightness further if persists

---

## Brightness Auto-Reset

**Problem**: Brightness resets to default when display wakes

### Workaround

- Check "Display Settings" for auto-brightness toggle
- May require manual adjustment on wake
- No permanent fix yet; INMO investigating

---

## Display Issues

### Ghosting/Double Vision

- **Cause**: Waveguide reflection with high-contrast content
- **Nature**: Inherent to 1D waveguide design
- **Workaround**: View external screens at angle or use one eye

### Rainbow/Shimmer Effect

- **Cause**: Vertical waveguide lines visible when viewing screens/bright objects
- **Mitigation**: Reduced with dark mode and proper screen positioning
- **Note**: Not removable, but acceptable to most users

### Home Bar Ghost Image

- **Cause**: Faint taskbar visible in corner due to reflection
- **Solution**: Use third-party launcher to hide stock bar

## Camera Issues

### Plug Removal

- Camera cover extremely recessed
- **Tools needed**: Toothpick, needle, or scotch tape
- Don't force; use small tool with care

### Shutter Sound

- Cannot disable in stock app
- Third-party camera apps may allow disabling

---

## Bluetooth Connectivity

### Issues

- Connection drops with multiple apps open
- Touchpad firmware updates improve reliability

### Solutions

- Close unused background apps
- Update touchpad firmware to v1.1.1
- Re-pair if persistent issues

## Casting/Mirroring Limitations

| Platform | Status |
|----------|--------|
| **iOS** | AirPlay works seamlessly |
| **Android** | Native Miracast not supported |
| **Solutions** | ApowerMirror (TV version), Duplila, or similar third-party apps |
| **PC** | Spacedesk (single display extension) |

See [Screen Mirroring](screen-mirroring.md) for detailed setup guides.

## Lock Screen Problems

**Issue**: Setting PIN code causes functional problems

- **Recommendation**: INMO advises against using lock screen features
- **Recovery**: Quick Settings app can remove PIN if locked out

---

## AI Assistant Connectivity

**Issue**: "Unstable network connection" error for US users

- **Status**: No ETA on fix from INMO
- **Workaround**: Use Google Lens or Gemini Live app directly

---

## Overheating (Chinese Version)

**Issue**: Extreme heat during video calls, navigation, remote desktop

**Cause**: Continuous background apps and less efficient cooling

### Solutions

1. Upgrade to international version if possible
2. Disable background apps
3. Add foam heat shields
4. Avoid charging while using intensive apps
