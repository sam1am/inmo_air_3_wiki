2. Download STL files
3. Print with appropriate materials (PLA/PETG)
4. Share your own designs with the community

## Software Mods

### YouTube ReVanced

- Ad-free YouTube experience
- Background playback support
- Sponsor block integration
- Download from ReVanced project

### Custom Launchers

- Improved home screen layouts
- Burn-in prevention (no static elements)
- Better app organization
- Recommended: Nova Launcher, Smart Launcher

### ADB Tweaks

Advanced customizations via Android Debug Bridge:

```bash
# Enable USB debugging first, then:
adb shell settings put system <setting> <value>
```

Common tweaks shared in Discord community.

## Safety Notes

- **Physical mods** may void warranty
- **Heat-related mods** should use appropriate materials
- **Bending components** requires care to avoid breakage
- **Test incrementally** before committing to permanent changes
