## iOS Mirroring (AirPlay)

- Works seamlessly and reliably
- Enables iPhone/MacBook screen casting to glasses
- Highly recommended for iOS users

### How to Use

1. Ensure glasses and iOS device are on same WiFi network
2. Open Control Center on iOS device
3. Tap Screen Mirroring
4. Select INMO Air 3

## Android Mirroring Solutions

### ApowerMirror (Recommended)

- **Version**: Use Smart TV version from Aptoide
- **Features**: Best option for Android casting
- **Quality**: Mirrors phone screen to glasses, maintains aspect ratio in landscape mode

### Duplila

- Works but with portrait mode borders in landscape
- Image quality good

### Native Android Casting

- **Miracast**: Not supported (no native implementation)
- Use third-party alternatives instead

### Samsung Devices

- **[Samsung Smart View](https://play.google.com/store/apps/details?id=com.casttotv.screenmirroring.smartview.smarttvlcd&hl=en_US)**: Can be used to share your device screen with Inmo's in-built screen sharing software. Enable "Chromecast" in Smart View's settings.

## PC/Mac Screen Mirroring

### Spacedesk (Windows only)

| Mode | Benefits |
|------|----------|
| **USB mode** | Lower latency |
| **WiFi mode** | More convenient |

- Single display extension via USB or WiFi
- Good for extending Windows desktop

### Chrome Remote Desktop (Recommended)

- Works with Windows, Mac, and Linux
- Removes screen borders for full visibility
- Good for productivity
- Free and reliable

### Remote Desktop Protocol (RDP)

- Built-in Windows Remote Desktop
- Requires configuration
- Best for LAN connections

### ApowerMirror

- Can mirror phone to glasses
- Not for PC mirroring (use other solutions)

## Game Streaming

| Service | Description |
|---------|-------------|
| **Moonlight** | Stream games from PC to glasses (low-latency, requires NVIDIA GPU) |
| **GeForce Now** | Cloud gaming service |
| **Xbox Game Pass** | Cloud gaming option |

### Moonlight Setup

1. Install Moonlight on glasses
2. Install GeForce Experience on PC
3. Enable GameStream in GeForce Experience
4. Pair devices on same network
5. Launch games from Moonlight app

---

## Tips for Best Experience

1. **Use 5GHz WiFi** for lower latency
2. **Chrome Remote Desktop** auto-adjusts borders for clean viewing
3. **Close unnecessary apps** on glasses to improve performance
4. **Use wired connection** (Spacedesk USB) when possible for lowest latency
5. **Adjust resolution** on source device if needed for readability
