## Battery & Power

| Specification | Value |
|---------------|-------|
| **Internal Battery** | 660mAh |
| **Battery Life** | 1.5-2 hours active use (video playback at 75% brightness) |
| **Charging Time** | 32 minutes (10% to 100%, powered off) |
| **Charging While Using** | ~1.1-1.2% per minute gain with 5000mAh power bank |
| **Bypass Charging** | Implemented (battery temp drops 1-2°C when plugged in) |
| **CPU Temp Change While Charging** | +3-4°C increase |

See [Battery & Power Management](battery-and-power.md) for detailed usage information.

---

## Temperature Management

| Condition | Temperature |
|-----------|-------------|
| **Idle** | ~98-99°F (37-38°C) |
| **Light Use** | ~104-108°F (40-42°C) |
| **Heavy Use** | Up to 115°F+ (46°C+) |
| **Chinese Version Issues** | Reported overheating to 137°F (58°C) during intensive tasks |

### Notes

- **International Version**: Improved cooling with copper heat dissipation pad
- **Critical Issue**: Using while charging (especially Google Lens, AnyDesk, navigation) can cause shutdown and app loss

See [Troubleshooting](troubleshooting.md) for heat management tips.
