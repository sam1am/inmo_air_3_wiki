| Control | Function |
|---------|----------|
| Directional pad | Navigation |
| Scroll | Finger drag |
| Click | Select |

| Anti-shake | Adjustable sensitivity |

## Prescription Lenses

| Aspect | Details |
|--------|---------|
| **Support** | Yes, full support including progressives |
| **Process** | Selection during Kickstarter campaign |
| **Quality** | Users report significantly clearer vision and less shimmer |
| **AR Coating** | Optional (adds cost, reduces reflections) |
| **Timeline** | 3-5 days stated production, often longer |

### Alternative Prescription Lenses

| Source | Details |
